Greenbro branding source of truth
=================================

- Approved originals live in `docs/branding/official/` (PDF/PNG wordmark + gear with tagline). Do not modify or regenerate the logo artwork.
- App-ready exports live in `mobile/assets/greenbro/`:
  - `greenbro-logo-horizontal.png` (tagline)
  - `greenbro-icon-1024.png` (gear on white)
  - `greenbro-splash.png` (horizontal logo centered on white)
- If new sizes are needed, export directly from the official artwork without redrawing, recoloring, or changing proportions. Keep transparent backgrounds for logos; keep white backgrounds for icon/splash.
- Legacy/incorrect assets are archived in `archive/branding-legacy/` (DO NOT USE).
