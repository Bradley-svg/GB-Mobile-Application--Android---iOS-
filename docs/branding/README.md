Greenbro branding source of truth
=================================

Use this file as the single reference for assets and colours.

Assets
------
- `mobile/assets/greenbro/greenbro-logo-horizontal.png` — horizontal logo with tagline.
- `mobile/assets/greenbro/greenbro-icon-1024.png` — app icon.
- `mobile/assets/greenbro/greenbro-splash.png` — splash artwork.
- `docs/branding/official/` holds the approved source artwork; export any new sizes directly from these originals.

Palette (from `mobile/app/theme/colors.ts`)
-------------------------------------------
- brandGreen `#39B54A`
- darker green `#2D9C3E` (gradient end)
- brandGrey `#414042`
- textPrimary `#111111`
- textSecondary / textMuted `#555555`
- background `#FFFFFF`; backgroundAlt `#F5F7F9`; brandSoft `#E9F7EC`
- borderSubtle `#E1E5EA`
- error `#DC2626`; warning `#D97706`; success `#16A34A`
- gradients: brandPrimary `#39B54A -> #2D9C3E`; brandSoft `#E9F7EC -> #FFFFFF`

Do not
------
- Do not recreate the logo as inline SVG text or custom shapes.
- Do not introduce new logo variants or change wording.
- Do not use or reintroduce the incorrect “GREEN BRO” logo/text.
- Do not add off-brand colours or gradients outside the defined theme.

Notes
-----
- Keep transparent backgrounds for logos; keep white backgrounds for icon and splash.
- Legacy/incorrect assets remain in `archive/branding-legacy/` for reference only (do not use).
